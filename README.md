# XP-inspired portfolio website

A static GitHub Pages-ready portfolio desktop.

## Use
1. Replace `Your Name` in `index.html`.
2. Replace `assets/curriculum-vitae.doc` with your actual CV.
3. Replace `assets/me.png` with your image.
4. Edit the `href` fields in `script.js` to point subitems to real project pages or external links.
5. Push all files to a GitHub repository and enable GitHub Pages from the repository settings.

## Notes
This design is inspired by early-2000s desktop interfaces and uses original CSS-drawn icons/backgrounds rather than Microsoft assets.
